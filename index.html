StyleableForm jQuery Plugin

Developed by @fikrirasyid

01. .styleableCheckbox() - Style checkbox easily

Styling Checkboxes One by One Using Their IDs:

$('#checkbox-1, #checkbox-2').styleableCheckbox();

Styling Dozen of Checkboxes Using Their Class:

$('.styled-checkbox').styleableCheckbox();

WARNING: the checkbox that want to be styled should have an ID for .styleableCheckbox() to work properly

02. .styleableTextPlaceholder() - More responsive placeholder

placeholder attribute of input[type="text"] rocks, but it only dissapears after the actual information is being typed on the input field. This plugin use value attribute's value as 'placeholder' and removes it as soon as the cursor is being focused on the text input field

Example:

$('#text-4').styleableTextPlaceholder();

  4. This is text input field which uses .styleableTextPlaceholder(). The Default Value dissapears right after the cursor is focused to the text input:

Make sure that you have included the type="text" attribute. It is required to make .styleableTextPlaceholder() works properly
